## KJs TRD Trading Terminal

## 🚀 Personal Trading Operating System

A professional trading platform built with Python, HTML, CSS, and JavaScript.

## ✨ Features

- ✅ Professional Trading Terminal UI
- ✅ Interactive Chart (Zoom, Pan, Hover)
- ✅ Multiple Timeframes (1m, 5m, 15m, 1h, 1d)
- ✅ Indicators (EMA, VWAP, RSI, MACD)
- ✅ Python Studio for Strategies
- ✅ Module Library
- ✅ Backend API (Flask)
- ✅ Database Support
- ✅ Buy/Sell Markers
